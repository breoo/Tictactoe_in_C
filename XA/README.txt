JOGO DA VELHA

Como funciona
O jogo é para 2 jogadores
Um joga com X e outro com O
Quem fizer 3 em linha ganha

Instrucoes
Clique em jogo da velha.exe para abrir
Digite a linha (0 a 2)
Digite a coluna (0 a 2)
X sempre comeca primeiro

O tabuleiro
As posicoes sao numeradas de 0 a 2

     0   1   2
   +---+---+---+
 0 | X | O | X |
   +---+---+---+
 1 | O | X |   |
   +---+---+---+
 2 |   | O | X |
   +---+---+---+

Formas de ganhar
3 em linha horizontal
3 em linha vertical
3 em qualquer diagonal

Se o tabuleiro ficar cheio sem vencedor e empate

Exemplo de jogada
Quando pedir Jogador X digite 1 1 para jogar no meio do tabuleiro

Desenvolvido em C
